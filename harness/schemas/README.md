# Harness Schemas

JSON schema contracts for events, checkpoints, policy, and logs live here.

Current schema files:

- `policy.schema.json`
- `event.schema.json`
- `checkpoint.schema.json`
- `todo.schema.json`
- `permission-denial.schema.json`

Notes:

- Schemas target JSON Schema Draft 2020-12.
- JSONL logs use one object per line, validated against the corresponding object schema.
